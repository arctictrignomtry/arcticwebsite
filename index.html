Getting started

Up and running in under a minute.

No dependencies to hunt down. No configuration files to edit.

1
Download Arctic
Grab the latest bootstrapper and install Arctic in a couple of clicks.
2
Launch Roblox
Open any Roblox game using the standard client, Bloxstrap, or Fishstrap.
3
Run Arctic and attach
Open Arctic.exe. Hit the attach button. Arctic detects the running client automatically.
4
Write and execute
Paste or write your Lua script in the editor. Press execute and watch it run.
